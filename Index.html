Описание приложения

Web Builder — это удобный инструмент для создания веб-страниц без необходимости программирования. Пользователь может добавлять элементы интерфейса, редактировать их внешний вид и сохранять результат.

Основные функции

  • Создание нового проекта: Нажмите кнопку 'Создать новый проект' для начала работы.
  • Добавление элементов: Используйте панель инструментов для добавления текста, кнопок, Navbar или Footer.
  • Редактирование элементов: Нажмите на элемент для изменения его параметров.
  • Сохранение и удаление: Сохраняйте или удаляйте элементы по необходимости.

Пользовательский интерфейс

  • Главный экран: Заголовок, кнопка для создания проекта и сохраненные проекты.
  • Рабочая область: Панель инструментов и белая область для работы.

Шаги использования

  1. Запустите приложение и начните новый проект.
  2. Добавьте элементы с панели инструментов.
  3. Редактируйте элементы через окно редактирования.
  4. Сохраняйте изменения или удаляйте элементы.
