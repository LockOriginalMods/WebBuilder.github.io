Описание приложения

Web Builder — это удобный инструмент для создания веб-страниц без необходимости программирования.

Основные функции

  • Создание нового проекта: Нажмите кнопку 'Создать новый проект'.
  • Добавление элементов: Панель инструментов позволяет добавлять текст, кнопки, Navbar и Footer.
  • Редактирование элементов: Нажмите на элемент для изменения параметров.
  • Сохранение и удаление: Сохраняйте или удаляйте элементы.

Пользовательский интерфейс

  • Главный экран: Заголовок, кнопка для нового проекта и сохраненные проекты.
  • Рабочая область: Панель инструментов и белая область для работы.

Шаги использования

  1. Начните новый проект.
  2. Добавьте элементы с панели инструментов.
  3. Редактируйте элементы в диалоговом окне.
  4. Сохраняйте проект или удаляйте элементы.
